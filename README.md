# AI Job Tracker

A full-stack application for tracking job applications with AI-powered features such as job analysis, match scoring, and cover letter generation.

Built as a portfolio project.

## Structure

- client — Next.js frontend
- server — backend API (Node.js)
- Monorepo managed with pnpm

## Getting Started

```bash
pnpm install
pnpm dev
```

Frontend: http://localhost:3000

Backend: http://localhost:3001

---

## Tech Stack

- Next.js
- React
- Node.js
- pnpm (monorepo)
- ESLint + Prettier
- Husky + lint-staged

## Scripts

- pnpm dev — run client and server
- pnpm build — build all packages
- pnpm lint — lint all code
- pnpm test — run tests

## Why this project

This project is built to practice full-stack development, clean architecture, and AI integration, and to serve as a portfolio project for junior full-stack / AI-focused roles.
